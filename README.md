# gopro.net

Minimal unofficial GoPro HERO 8 API library for .net

# Using 

Example 

Upload last file 

```sh
asv-gorpo.exe last -h=10.10.4.83 -p=80 -t=60000 
```

For proxy

```sh

sudo socat TCP-LISTEN:80,fork TCP:10.5.5.9:8080

```
